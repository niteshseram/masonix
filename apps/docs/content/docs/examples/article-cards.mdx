measures the rendered card and places the next item in the shortest column.

## When to use

Use this for blogs, resource hubs, changelogs, case studies, and dashboard
cards with optional sections.

## Notes

- Pick an `estimatedItemHeight` near the average rendered card.
- Use stable keys so measured heights follow the correct article.
- Use `minItemHeight` if loading states can briefly measure too small.
