## When to use

Use this when the API already gives image dimensions. It works well for
portfolio pages, editorial galleries, and commerce lookbooks.

## Notes

- Known heights reduce layout shift.
- `defaultWidth` improves the first server-rendered layout.
- Keep `alt` text on the actual image, not on the wrapper.
