Masonix renders placeholders that keep the same layout size.

## When to use

Use this when cards include expensive media, charts, syntax highlighting, or
heavy interactive controls.

## Notes

- Placeholder height should be used directly.
- Keep placeholders visually quiet.
- Real item keys, positions, and measurements remain intact.
