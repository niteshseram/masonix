navigation, or a "back to top" action.

highlighted after the control runs.

## When to use

Use this for search navigation, selected-item focus, keyboard shortcuts,
timeline jumps, and admin tools.

## Notes

- `align` can be `start`, `center`, or `end`.
- `smooth` uses browser smooth scrolling.
- For custom panels, also pass `scrollContainer`.
