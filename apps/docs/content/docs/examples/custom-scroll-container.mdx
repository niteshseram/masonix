## When to use

Use this for dashboards, modals, side panels, docs previews, and embedded app
surfaces.

## Notes

- The ref should point to the element that actually scrolls.
- Keep the scroll container height constrained.
- Use `min-h-0` in CSS grid/flex layouts so the panel can scroll.
