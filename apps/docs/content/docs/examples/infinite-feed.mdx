## When to use

Use this when a feed is too large to render all at once and data arrives in
pages.

## Notes

- `endReachedThreshold` is measured in remaining loaded items.
- `onEndReached` fires once per loaded item count.
- Pass `totalItems` when the loaded count differs from the known total count.
